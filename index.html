Overview

The project redesigns a classic small-scale electromagnetic accelerator that speeds up a magnetic ball around a track. Serving as a visual demonstration of Faraday's law of Electromagnetic induction, the prototype effectively delivers power to maximize work. It incorporates powering copper coils into a microelectronic system controlled by software algorithms to control the speed of an accelerating ball.

Role & Challenges

I served as the lead hardware designer, responsible for creating the electrical design, sourcing the necessary components, and assembling the final prototype. Key challenges included physical wear that compromised the continuous motion of the ball, power switching between the coils, and integrating the microcontroller into the system.

Technologies & Skills

The project utilizes Python for controlling the input signals from the different sensors in the system, and for sending the output signals for power switching. The hardware required soldering to create the final electrical system. The overall project required teamwork and adaptability to navigate different component sourcing complications and physical limitations.

Outcome

The project's result enabled successful control of the ball around the track. Power and physical constraints limited the system's ability to control the exact speed. The project highlighted the complicated process of going from calculations and design to a fully implemented prototype.

Overview

The project aims to expand accessibility in the realm of control by providing a proof-of-concept prototype for an open-source custom brain control interface. Over the course of a semester, this senior-level design project developed a homemade EEG reader to detect and filter brainwaves, along with accompanying filtering software that enabled users to play a Pong game.

Role & Challenges

I served as the lead hardware designer, responsible for creating the custom analog filter, simulating it, and verifying its functionality in the lab. Additionally, I assembled the final hardware system and soldered its distinct components. Key challenges in this project were limited documentation for establishing the EEG reader and bridging the gap between non-ideal components and expected functionality to create a reliable system.

Technologies & Skills

The project relied on circuit theory and design principles to create the necessary filtering circuit. With SPICE simulation software, designs were validated and optimized. Using electrical lab equipment such as oscilloscopes, multimeters, and soldering kits, the prototype was extensively debugged, tested for functionality, and assembled.

Outcome

The project provided proof of concept that our system could create control functions using our brain waves. By monitoring attention levels, the system allowed the user to play a Pong game. Working under an agile workflow, our team applied what we learned between each iteration to adapt and overcome the continually emerging engineering challenges.

Overview

The project aims to create a multiplier peripheral at the gate level and develop a driver capable of communicating with a Linux kernel on an FPGA to implement the multiplication function at the hardware level.

Role & Challenges

I created the hardware peripheral, driver, and implemented the Linux kernel onto the FPGA. The biggest challenge was the time it took to make the Linux kernel and upload it to the FPGA. Any error would result in having to start the process over from the beginning.

Technologies & Skills

The project relied on Verilog to create the hardware peripheral, along with the test benches to verify its functionality. Using an FPGA-based microprocessor, the multiplication peripheral, along with the Linux kernel, is implemented. The C language is then used to communicate between the kernel and peripherals, allowing for the multiplication functionality at the gate level.

Outcome

The project demonstrated a joint software and hardware approach. This basic demonstration illustrated the process of creating gate-level peripherals and how a microprocessor system can access them using embedded languages such as C, thereby reducing the amount of work done by the CPU.

Overview

The project aims to create an 8-bit adder at the transistor level and create the material layout. The project also aims to optimize timing and power consumption by managing size and distribution.

Role & Challenges

I created each layout, gate schematic, and symbol for each component within the 8-bit adder. I also sized each logic gate and performed timing analysis and power consumption checks for each element. The most challenging part was getting accustomed to the software tools and having the patience to test and analyze each component and stage thoroughly.

Technologies & Skills

The project relied on a digital circuit design utilizing Cadence Virtuoso to create and simulate the schematics. Optimization techniques, including timing analysis and power consumption checks, guided the sizing and distribution of the layouts.

Outcome

The project created a thoroughly tested and optimized 8-bit adder. It shows the process from creating each gate at the transistor level to a consecutively larger component, illustrating the process of Very Large Scale Integration at a smaller scale.

Overview

The project aims to interface microcontrollers with A/D converters using an ADC chip to analyze analog signals and characterize the result.

Role & Challenges

I helped interface the microcontroller with the ADC chip using a Serial Peripheral Interface communication bus. Additionally, I assisted in developing a method for characterizing the output by utilizing threshold values to identify the wave type. The challenge with this project was calibrating the code to match the actual input more closely, along with characterizing the input wave.

Technologies & Skills

The project utilized Python to create the connection between the microcontroller and ADC. The project used the SPI communication protocol to synchronize communication between the ADC and microcontroller, enabling the collection and characterization of data.

Outcome

The resulting project successfully detected the amplitude, frequency, and wave type of an input wave.
